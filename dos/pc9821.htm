初代PC-9821にして唯一のWindows 3.0プリインストールモデル。
この記事ではWindows 3.0A-H / MME 1.0-H 以外の事項について述べる。

★MS-DOS 5.0A-H★
・製品番号 0104 , リビジョン 0004
　製品版でいうところの黄シール版に相当
　手持ちの黄シール版と比較したところ、
　PC-9821添付の5.0A-Hのほうが古いという意外な結果に

・製品版と異なり、AVSDRV.SYSとAVGDRV.SYSが標準収録

・PC-9821にはCDドライブが搭載されているがDOSのディスクに
　CDドライブのドライバ類がなく、使用するためには
　Windows MME をセットアップする必要がある

・PC-9821/S1のMS-DOS 5.0A-Hは5枚組らしい

・初回起動時に起動する「固定ディスクチェックプログラム」は
　セットアップFDに収録されていない

★メニューファイルについて★
・MS-DOS ディスク#1にはMENU.MNUが収録されているが、
　セットアップでコピーされないので手動コピーが必要

・MENUから呼び出されるCD起動用バッチファイル群は
　Windows 3.0A-H ディスク #1に収録されており、
　COPYCD.BAT を実行することでインストールが可能

★PC-9821/S2の設定★
出荷時の設定ファイルをメモとして置いておきます

・CONFIG.SYS
FILES=30
BUFFERS=10
SHELL=A:\COMMAND.COM /P
DEVICE=A:\DOS\HIMEM.SYS
DEVICE=A:\DOS\SMARTDRV.SYS 2048 512
DOS=HIGH
DEVICE=A:\WINDOWS\NECCD.SYS /D:CD_101

・AUTOEXEC.BAT
@ECHO OFF
A:\WINDOWS\MSCDEX /D:CD_101
PATH A:\WINDOWS;A:\DOS;A:\
SET TEMP=A:\DOS
SET DOSDIR=A:\DOS
WIN
MENU
