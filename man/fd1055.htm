FD1055

5" 2DD専用ドライブ(※)、VFOなし
PC-9801VF内蔵FDD; 実は2HD/2DD両用ドライブ		 

ターミネータDIP抵抗330Ωが搭載
PC-9801VFではドライブ2に搭載

制御基板はG9VLT 134-835274
HMC-503AというハイブリッドICがとても目立つ
85/8, 85/10に製造されたFD1155Cと同じ制御基板
86年製造のFD1155Cは同じ型番だが使用ICが異なる

USE: PIN4割当 VF既定値は1
1=HEAD LOAD, 2=IN USE

DX: DRIVE SELECT割当 (1 or 2)
0=DS0, 1=DS1, 2=DS2, 3=DS3, MX=常に選択状態

DCG: PIN34割当 VF既定値は1
1=READY 2=DISK CHANGE

MON: モータオン条件 VF既定値は1
1=MOTOR ON, 2=HEAD LOAD, 3=DRIVE SELECT

HS: ヘッドロード条件 VF規定値は1
1=DRIVE SELECT有効時 2=DRIVE SELECT無視

LED: LED点灯条件　VF既定値は1&5
1-3と4-5それぞれどちらかをショートで選択
1&4 = HEAD LOAD
1&5 = HEAD LOAD & DRIVE SELECT
2&4 = DRIVE SELECT
2&5 = DRIVE SELECT
3&4 = READY
3&5 = READY & DRIVE SELECT

HDE: 2DD時の回転速度　VF既定値は2
 1=360rpm 2=300rpm

DEN: 密度切替設定 VF既定値はOPEN
1 = PIN2で切替;
2 = PIN2で切替; 1と論理反転
3 = 2HD固定
OPEN = 2DD固定
